# App Assets

This directory contains the app icons and splash screens.

## Required Files

- **icon.png** (1024x1024) - App icon for iOS and Android
- **adaptive-icon.png** (1024x1024) - Android adaptive icon foreground
- **splash.png** (2048x2048 recommended) - Splash screen image
- **favicon.png** (256x256) - Web favicon

## Current Status

⚠️ **Placeholder images are currently in use.** These are minimal valid PNG files that allow the build to proceed, but you should replace them with proper app icons before releasing your app.

## How to Replace

1. Design or obtain your app icons in the required sizes
2. Replace the placeholder files in this directory
3. Ensure all images are PNG format
4. Rebuild your app

## Recommended Tools

- [App Icon Generator](https://www.appicon.co/)
- [Favicon Generator](https://www.favicon-generator.org/)
- Design tools: Figma, Canva, Adobe Illustrator, etc.

## Icon Guidelines

- **App Icon**: Should be 1024x1024 PNG, no transparency for iOS
- **Adaptive Icon**: 1024x1024 PNG, safe zone is central 66% (Android)
- **Splash Screen**: 2048x2048 PNG recommended, centered design
- **Favicon**: 256x256 PNG for web

## Colors

The app uses green (#4CAF50) as the primary color, which matches the splash screen background.
